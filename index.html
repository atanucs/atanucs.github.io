Publications

Atanu Kundu, Sarthak Das, Rajarshi Ray. SAT-Reach: A Bounded Model Checking for Affine Hybrid Systems. ACM Transactions on Embedded Computing Systems. 22(2), January 2023.

In this paper, we proposed a BMC algorithm based on the CEGAR framework. The foremost step of the algorithm is obtaining an abstract counterexample from the discrete structure of the HA. Then two-tier refinements are carried out to produce a concrete counterexample. The abstract counterexample,which is a simple path (alternative sequence of locations and transitions) obtained using SAT-based graph encoding of the HA. The reachability analysis is performed along the paths as a first round refinement, if the reachable states are interesected with forbidden region, we apply the final refinement that is the trajectory splicing algorithm to find a concrete counterexample from the overapproximated abstract counterexample. We have shown the efficacy of our algorithm by comparing with exsisting tool such as dReach, XSpeed, Flow*, SpaceEX. Our algorithm is implemented in a tool SAT-Reach.

Hybrid Systems

The systems which are \emph{hybrid} in nature are called \emph{hybrid systems} or sometimes they are also known as \emph{cyber physical systems}. Hybrid systems consist of both continuous and discrete dynamics behavior, such type of systems can arise when a digital controller interacts with the physical world via inputs and outputs, like aircraft flight control system. Nowadays such systems are developing rapidly and showing its immense use everywhere, from smart buildings to medical devices to automobiles. Building a reliable hybrid system is different from the process of developing traditional systems and the latter is of no direct help to build former due to its leading characteristics: \emph{real-time computation}, \emph{safety-criticality}, etc. An example of hybrid systems is a cruise controller of a car.

Formal Verifications

Formal verification is a technique used to prove or disprove the correctness of a system or program using mathematical methods. It involves defining formal specifications of the system's behavior and using formal methods such as theorem proving, model checking, and static analysis to verify if the system satisfies the specifications. Formal verification can help detect and eliminate errors, improve system reliability and safety, and reduce the cost of testing and debugging. It is widely used in safety-critical systems such as avionics, automotive, and medical devices. However, formal verification can be computationally expensive and requires expertise in formal methods and software engineering.

Bounded Model Checking

Bounded model checking (BMC) is a technique used in formal verification to verify whether a finite-state system satisfies a given property. It involves constructing a finite-state model of the system, specifying the property to be verified in temporal logic, and then checking if the property holds for all possible execution paths of the system up to a certain length, or "bound". If the property holds for all bounded execution paths, then it is concluded that the system satisfies the property for all possible executions. BMC has several advantages over other verification techniques, such as being more efficient than exhaustive model checking and being able to handle complex properties.

Machine Learning

Machine learning is a subfield of artificial intelligence that focuses on developing algorithms and models that enable computers to learn from data and improve their performance on a task without being explicitly programmed. Machine learning involves identifying patterns in data through statistical analysis, and then using those patterns to make predictions or decisions.
